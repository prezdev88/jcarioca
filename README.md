### ✅ Completado

- Sistema de barajas y mazos
- Mezclador de cartas
- Reparto de cartas a jugadores
- Turnos entre jugadores
- Sistema de juego por consola

### 🔄 En Progreso

- Validación de combinaciones (tríos, escaleras)
- Sistema de puntos
- Condiciones de victoria

### 📋 Por Hacer

- Interfaz gráfica (GUI)
- Modo multijugador online
- Tests unitarios
- Documentación completa

---

## 🕹️ Cómo Jugar

```bash
# Compilar el proyecto
mvn compile

# Ejecutar el juego
mvn exec:java
```

### Controles

| Opción | Acción |
|--------|--------|
| 1 | Ver montoncito (descarte) |
| 2 | Tomar carta del montoncito |
| 3 | Tomar carta del mazo |
| 4 | Jugar una carta |
| 5 | Ver mis cartas |
